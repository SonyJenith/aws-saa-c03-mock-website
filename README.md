# AWS SAA-C03 Mock Console

**A free, timed mock-exam simulator for the AWS Certified Solutions Architect – Associate exam — 390 real practice questions, full explanations, zero sign-up.**

### 🔗 [sonyjenith.github.io/aws-saa-c03-mock-website](https://sonyjenith.github.io/aws-saa-c03-mock-website/)

---

## Why this exists

Most SAA-C03 practice question banks give you the answers up front, which makes it impossible to actually test yourself. This tool hides every answer until you submit — then breaks down exactly what was right, what wasn't, and why, question by question.

## What's inside

- **6 full-length practice tests** — 65 questions each, 390 total
- **Exam Mode** — 130-minute countdown, answers hidden until submit, question navigator with mark-for-review, just like the real test
- **Practice Mode** — untimed, reveals the explanation immediately after each question
- **Detailed reviews** — every question includes the correct answer(s) and a full explanation of why each option is right or wrong
- **Score breakdown** — pass/fail against the real 72% bar, with filters to review only what you missed or flagged
- **No installs, no accounts, no tracking** — it's a single static HTML page

## Try it

👉 **[Launch the mock console](https://sonyjenith.github.io/aws-saa-c03-mock-website/)**

Pick a test, choose Exam or Practice mode, and go. Progress lives only in your browser tab for that session, so try to finish a test in one sitting.

## Tech

A single self-contained `index.html` — no build step, no dependencies, no backend. Just HTML, CSS, and vanilla JavaScript, hosted for free on GitHub Pages.

## Running it locally

```bash
git clone https://github.com/sonyjenith/aws-saa-c03-mock-website.git
cd aws-saa-c03-mock-website
open index.html   # or just double-click it
```
